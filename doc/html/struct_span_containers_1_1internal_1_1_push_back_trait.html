Assign values to the back of the container.

This method only provides the basic exception gurantee. If range exceeds the capacty of the container, values will be pushed_back until capcity is reached, at which point a FullContainerError exception is thrown.

Template Parameters
RangeThe type of the range that contains the values.
Exceptions
FullContainerErrorIf the size of the range exceeds the container capacity and UseExceptions is true.
